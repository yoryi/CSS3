Que es CSS3

Como funciona?

Css funciona directamente con el html, es decir cuando solicitamos nuestro index o una pagina web, directamente el color o estilo que se halla puesto lo mostara el navegador.

Estilos Internos

se le llaman estilos internos, aquellos estilos que se utilizan directamente dentro del html, cuya cabecera se realiza la decalariones y formatos que se desea aplicar en la paguina web.

Estilos Externos

Se le llaman estilos externos, aquellos estilos que se encuentran en una direcion o ruta externa del html, es decir estos estilos pueden estar dentro de una carpeta renombrados con un nombre y una extencion .css

Selectores CSS

Los selectores de css son muy importantes, ya que de esta forma declaramos un estilo unico o global para cada vista sea por medio de clases o id.

ID

Los selectores ID se declara con un # seguido de el nombre de ese identificador.

Ejemplo

#hola{

size: 14

padding-left: 2%;

}

CLASS

Los selectores CLASS se declara con un . seguido de el nombre de ese identificador.

Ejemplo

.hola{

size: 14

padding-left: 2%;

}
